# critical_analysis_of_CRPD_alternative_reports
This repository contains the code and documentation for a mixed-methods computational text analysis project examining civil society alternative reports submitted to the UN Committee on the Rights of Persons with Disabilities (CRPD). Using a corpus of 337 English-language submissions collected via Selenium automation from the OHCHR Treaty Body Database, the project applies term frequency analysis, TF-IDF, bigram analysis, GloVe word embeddings, Named Entity Recognition (spaCy), lexicon-based sentiment analysis (Bing), and LLM-assisted qualitative coding (NotebookLM) to identify policy priorities, regional variation, and intersectional vulnerability patterns across civil society reporting. All analysis is conducted in R and Python within a Quarto literate programming environment.
